# E-Commerce Sales Analytics Dashboard

## Project Description
An end-to-end data analytics project focused on analyzing e-commerce sales data to uncover revenue trends, top-performing product categories, high-revenue cities, and customer purchase patterns.

## Objective
To support data-driven decision-making by transforming raw sales data into meaningful insights using Python and visualizing them through an interactive Power BI dashboard.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- CSV Dataset

## Key Highlights
- Cleaned and preprocessed raw e-commerce sales data
- Performed feature engineering for time-based and business metrics
- Conducted exploratory data analysis (EDA) to identify trends and patterns
- Built an interactive Power BI dashboard with KPIs and filters

## Dashboard Insights
- Total Revenue, Total Orders, Average Order Value
- Monthly revenue trends
- Category-wise sales performance
- City-wise revenue distribution
- Weekday sales patterns

## Files Included
- `notebooks/` – Python notebook for data cleaning, feature engineering, and EDA
- `dashboard/powerbi_dashboard.pbix` – Power BI dashboard file
- `dashboard/dashboard_screenshot.png` – Dashboard preview image

## Dataset
The dataset used in this project is a large e-commerce sales dataset and is not uploaded to this repository due to GitHub file size limitations.  
The cleaned dataset was generated using Python.

## Conclusion
This project demonstrates a complete data analytics workflow from data preparation and analysis in Python to building a business-ready dashboard in Power BI.
